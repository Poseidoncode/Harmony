# Harmony Prompt Assistant

一個基於 Vue 3 的 VS Code 提示詞助手擴充功能。

[English Version](./README.md)

---

## 使用說明 (繁體中文)

### 1. 開啟 Harmony
點擊 VS Code 側邊活動列（Activity Bar）中的 **Harmony 圖示** 即可開啟側邊欄。

### 2. 模板管理
- **編輯模板**: 點擊 **「編輯模板」** 按鈕。這會直接開啟 `templates.json` 檔案，儲存後的變更將自動同步到側邊欄。
- **匯入 (取代)**: 使用 **「匯入 (取代)」** 按鈕從 JSON 檔案載入全新模板並取代現有內容。
- **匯入 (增補)**: 使用 **「匯入 (增補)」** 按鈕將 JSON 檔案中的模板加入目前清單的開頭。
- **刪除**: 點擊模板旁的垃圾桶圖示可直接刪除該模板。

### 3. 提示詞操作
- **插入**: 點擊模板名稱即可將內容插入當前編輯器或終端機。
- **Chat (執行)**: 直接將提示詞發送到 **GitHub Copilot Chat** 並立即執行。
- **Chat (草稿)**: 將提示詞填入 **Copilot Chat** 輸入框作為草稿，不立即執行。

---

## 功能特色
- **自動同步**: 透過擴充功能介面開啟並編輯儲存模板檔案後，Sidebar 會自動刷新，無需重啟。
- **持久化儲存**: 您的模板現在儲存在 VS Code 的全域儲存空間中，擴充功能更新後仍會保留。
- **快速插入**: 提供多種模板，支援直接進插入編輯器或發送到 GitHub Copilot Chat。
